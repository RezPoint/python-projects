# Python Projects: Обучение и Data Science

Это репозиторий с примерами и уроками по Python для изучения с нуля и перехода к Data Science.  
Проект структурирован по **этапам**, чтобы обучение было последовательным.

---

## Структура репозитория

### Этап 1: Основы Python
Папка: [01_basics_python](01_basics_python/)  
Содержит уроки по основам Python:

- [01_hello](01_basics_python/01_hello/) — Первая программа "Привет, мир!"  
- [02_variables](01_basics_python/02_variables/) — Переменные и типы данных  
- [03_user_input](01_basics_python/03_user_input/) — Ввод данных от пользователя  
- [04_loops](01_basics_python/04_loops/) — Циклы `for` и `while`

### Этап 2: Data Science
Папка: [02_data_science](02_data_science/)  
Содержит примеры по обработке данных и статистике:

- [data_statistics.py](02_data_science/data_statistics.py) — пример работы с данными, базовые вычисления и статистика
